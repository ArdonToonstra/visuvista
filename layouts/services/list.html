{{ if eq $lang "es" }} De escaneo a visualización: 6 fases para resultados perfectos {{ else if eq $lang "nl" }} Van scan tot visualisatie: 6 fasen voor perfecte resultaten {{ else }} From scan to visualization: 6 phases for perfect results {{ end }}

{{ if eq $lang "es" }} Fase 1: Escaneo 3D (Situación Actual) {{ else if eq $lang "nl" }} Fase 1: 3D Scanning (Huidige Situatie) {{ else }} Phase 1: 3D Scanning (Current State) {{ end }}

{{ if eq $lang "es" }} Escaneamos in situ toda la propiedad y jardín con escáneres 3D de alta precisión, creando una 'nube de puntos' - una copia digital exacta del estado actual. {{ else if eq $lang "nl" }} We beginnen op locatie om de gehele woning en tuin in te meten met 3D scanners, wat resulteert in een 'Point Cloud' - een extreem nauwkeurige digitale kopie van de huidige staat. {{ else }} We start on-site to measure the entire property and garden with 3D scanners, resulting in a 'Point Cloud' - an extremely accurate digital copy of the current state. {{ end }}

{{ if eq $lang "es" }} Fase 2: Modelado 3D (Estado 'As-Is') {{ else if eq $lang "nl" }} Fase 2: 3D Modellering (Bestaand 'As-Is') {{ else }} Phase 2: 3D Modeling (Existing 'As-Is') {{ end }}

{{ if eq $lang "es" }} La nube de puntos se procesa en un modelo 3D 'As-Is' preciso, que sirve como base para el nuevo diseño y previene malentendidos por errores de medición. {{ else if eq $lang "nl" }} De Point Cloud wordt verwerkt tot een nauwkeurig 3D 'As-Is'-model. Dit model dient als basis voor het nieuwe ontwerp en voorkomt misverstanden door meetfouten. {{ else }} The Point Cloud is processed into an accurate 3D 'As-Is' model. This model serves as the foundation for the new design and prevents misunderstandings due to measurement errors. {{ end }}

{{ if eq $lang "es" }} Fase 3: Modelado 3D (Nuevo 'To-Be') {{ else if eq $lang "nl" }} Fase 3: 3D Modellering (Nieuw 'To-Be') {{ else }} Phase 3: 3D Modeling (New 'To-Be') {{ end }}

{{ if eq $lang "es" }} Basándose en el modelo 'As-Is', diseñamos la nueva situación. Se procesan todas las especificaciones: muros digitalmente removidos, extensiones modeladas, escaleras nuevas y diseño completo de jardín. {{ else if eq $lang "nl" }} Op basis van het 'As-Is'-model wordt de nieuwe situatie ontworpen. All specifications are processed: walls digitally removed, extensions modeled, new stairs and complete garden and pool design. {{ end }}

{{ if eq $lang "es" }} Fase 4: Presentación Preliminar {{ else if eq $lang "nl" }} Fase 4: Voorlopige Presentatie {{ else }} Phase 4: Preliminary Presentation {{ end }}

{{ if eq $lang "es" }} Momento de revisión in situ: recorremos juntos el modelo 3D para verificar que todos los cambios se hayan implementado correctamente antes de la renderización final. {{ else if eq $lang "nl" }} Revisie moment op locatie: we lopen samen door het 3D model om te controleren of alle wijzigingen correct zijn doorgevoerd voordat we definitief gaan renderen. {{ else }} On-site review moment: we walk through the 3D model together to check that all changes have been implemented correctly before final rendering. {{ end }}

{{ if eq $lang "es" }} Fase 5: Visualización, Ambiente y Luz {{ else if eq $lang "nl" }} Fase 5: Visualisatie, Sfeer & Licht {{ else }} Phase 5: Visualization, Atmosphere & Lighting {{ end }}

{{ if eq $lang "es" }} Con el modelo 'To-Be' listo, añadimos materiales finales, iluminación global y ambiente extra para crear visualizaciones fotorrealistas. {{ else if eq $lang "nl" }} Wanneer het 'To-Be'-model gereed is, worden de uiteindelijke materialen, globale verlichting en extra sfeer toegevoegd voor fotorealistische visualisaties. {{ else }} When the 'To-Be' model is ready, final materials, global lighting and extra atmosphere are added for photorealistic visualizations. {{ end }}

{{ if eq $lang "es" }} Fase 6: Presentación Final {{ else if eq $lang "nl" }} Fase 6: Eindpresentatie {{ else }} Phase 6: Final Presentation {{ end }}

{{ if eq $lang "es" }} Ronda final de feedback presencial sobre detalles y elección de materiales, seguida de la entrega definitiva con renders fotorrealistas de alta calidad. {{ else if eq $lang "nl" }} Laatste feedbackronde op locatie voor details en materiaalkeuzes, gevolgd door definitieve oplevering met hoogwaardige fotorealistische renders. {{ else }} Final feedback round on-site for details and material choices, followed by definitive delivery with high-quality photorealistic renders. {{ end }}
